# Raaya Social – Single Page Website

Raaya Social is a **modern responsive single-page website** built using **HTML, CSS, and Vanilla JavaScript**.  
The project showcases a creative service platform that provides **short-form video content for businesses and personal brands**.

The website includes multiple sections such as services, portfolio, client reviews, FAQ, and contact form, all structured in a **clean single-page layout** with responsive design.

---

## 🌐 Live Demo

View the live website here:

👉 https://rayasocial.vercel.app/

---

## 🚀 Project Overview

Raaya Social presents a marketing landing page for a creative agency offering:

- Short-form video content
- Social media content creation
- Creative service packages
- Client testimonials
- FAQ section
- Contact form

The layout is optimized for **desktop, tablet, and mobile devices**.

---

## 🛠 Tech Stack

This project is built with:

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling, layout, and responsiveness  
- **Vanilla JavaScript** – Interactivity (menu toggle, FAQ accordion)  
- **Google Fonts** – Typography  

Fonts used:

- **Poppins**
- **Raleway**
